# Brand Templates Playbook — Universal
## Version: 1.0 | Last Updated: 2026-02-20

## How Brand Templates Work
Each business defines its visual identity in brand.md:
- primary_colour (hex)
- accent_colour (hex)
- font_heading
- font_body
- logo_path (in /businesses/{name}/assets/)
- logo_placement (top-left | top-right | bottom-left | bottom-right)

Puppeteer loads these values and renders branded graphics automatically.

## Template Types (generated for every business)
1. **Carousel slide** — bg + text + image area + logo
2. **Before/after split** — divider + labels + logo
3. **Stat card** — large number + context + brand colours
4. **Testimonial card** — quote + stars + name + brand frame
5. **Listing graphic** — details + image + price + CTA
6. **Ad creative** — hero image + headline + CTA button + trust signal
7. **Story slide** — full-bleed image + text overlay + brand bar

## Consistency Rules
- Same brand treatment across all platforms
- Logo never distorted or recoloured
- Fonts consistent within business (never mix)
- Colour palette limited to primary + accent + white + black/dark
